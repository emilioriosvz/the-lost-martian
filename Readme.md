## Getting Started
Clone the repo:
```sh
git clone git@github.com:emilioriosvz/the-lost-martian.git
cd the-lost-martian
```

Install dependencies:
```sh
npm i
```
or

```sh
yarn
```

Run:
```js
npm start
```

Test:
```js
npm test
```
